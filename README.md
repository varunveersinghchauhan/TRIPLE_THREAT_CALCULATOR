# ⚡ TTC Pro — Triple Threat Calculator

A futuristic, all-in-one **web-based calculator** that combines:

* 🧮 Scientific Calculator
* 📈 Financial Calculator
* 💪 BMI Calculator

Designed with **modern UI/UX**, glassmorphism, and smooth animations for a premium user experience.

---

## 🚀 Live Demo

👉 [GitHub Pages link here](https://varunveersinghchauhan.github.io/TRIPLE_THREAT_CALCULATOR/)

---

## ✨ Features

### 🧮 Scientific Calculator

* Advanced math operations (sin, cos, tan, log, √, factorial, etc.)
* Real-time result preview
* Expression history with recall
* Keyboard shortcuts support
* Clean and responsive UI

---

### 📈 Financial Calculator

* EMI Calculator
* Future Value Calculator
* Compound Interest
* Simple Interest
* Interactive sliders for input
* Graph visualization using charts

---

### 💪 BMI Calculator

* Metric & Imperial units support
* Real-time BMI calculation
* Visual BMI scale (needle indicator)
* Health category detection
* Smart health tips

---

## ⚙️ How to Run Locally

1. Download or clone the repository:

```
git clone https://github.com/your-username/your-repo-name.git
```

2. Open the folder
3. Double-click `index.html`
   OR open with Live Server

---

## 🌐 Deployment (GitHub Pages)

1. Go to repository → Settings
2. Click **Pages**
3. Select:

   * Source → Deploy from branch
   * Branch → `main`
4. Save

Your site will be live at:

```
https://your-username.github.io/repo-name/
```
